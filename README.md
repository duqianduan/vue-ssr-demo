
# vue-ssr-demo
The ssr solution based on vue2 with vue-router and vuex. 

### Install
``` bash
# download from github
git clone https://github.com/duqianduan/vue-ssr-demo.git
# cd project
cd vue-ssr-demo
# install
npm install
```
### Run
``` bash
# build bundle and start node server
npm run dev
```
